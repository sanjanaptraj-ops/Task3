# Phishing Email Detection Model

## Overview
This project uses Machine Learning and Scikit-learn to detect phishing emails based on textual content and URL patterns.

## Features
- Email preprocessing
- URL detection
- TF-IDF feature extraction
- Phishing/Safe email classification
- Accuracy evaluation
- Confusion Matrix visualization

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python phishing_detector.py
```

## Output
- Classification of email as Safe or Phishing
- Accuracy score
- Confusion matrix graph
